## G15 + G17 — multiple-testing + meta-policy

Every subgroup analysis MUST report `multiple_testing_correction` with the
method, n_tests, and adjusted alpha. Subgroups whose p-value fails the
adjusted threshold are downgraded to `claim_layer: "exploratory"` even if
the raw p is < 0.05.

`interaction_p_value` (effect-modification test) is REQUIRED for any
subgroup whose median is materially different from the primary cohort.

## Empty-integrator rule

If any subgroup's lifelines result is missing (cohort too small), record
the subgroup but set its `median_months: null`, `claim_layer: "speculative"`,
and add `note: "n too small per G15/G17 prereq"`.
