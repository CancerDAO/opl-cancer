# OPL for Cancer — AI Scientist Team Skill

> **Status: P0 Skeleton.** No experts / task packages / integrators implemented yet. This skill loads but performs no real work.

## How it will work (post-P0)

1. Initializes PI (Sid) for the patient
2. Sid receives natural language input and decides which experts from the 18-name roster to engage
3. Each engaged expert runs its internal 6-primitive grammar
4. Auditor "Henry" enforces IRB-substitute 4-layer transparency + Level 0-4 permission gates
5. Sid delivers consolidated patient brief with three-tier labels, PMID-linked citations, and provenance hash per claim
6. Patient is the sole decision authority — no human-in-the-loop external sign-off
